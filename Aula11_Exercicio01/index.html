Exemplo de Cores de Texto do Bootstrap

  • Texto Principal - A classe .text-primary é utilizada para indicar um texto importante ou a cor principal do tema[cite: 158].
  • Texto Secundário - A classe .text-secondary é para texto secundário[cite: 163].
  • Texto de Sucesso - A classe .text-success indica uma ação bem-sucedida[cite: 159].
  • Texto de Informação - A classe .text-info representa alguma informação[cite: 160].
  • Texto de Aviso - A classe .text-warning é usada para representar um aviso[cite: 161].
  • Texto de Perigo - A classe .text-danger representa perigo[cite: 162].
  • Texto Suave - A classe .text-muted é para um texto mais suave ou menos proeminente[cite: 157].
  • Texto Escuro - A classe .text-dark representa um texto cinza escuro[cite: 164].
  • Texto Padrão - A classe .text-body define a cor padrão do corpo do texto[cite: 165].
  • Texto Branco - A classe .text-white é usada para texto branco, e neste exemplo, um fundo escuro foi adicionado para visualização[cite: 156, 172].
